# Barber Appointment Service

Pet-проект: веб-сервис для записи на стрижку с разграничением доступа (роли клиент, мастер, администратор).  
Разработан для демонстрации навыков backend-разработки на Python (Flask) с использованием современных подходов: ORM, миграции, аутентификация, REST API, unit-тесты, кастомные декораторы прав.

## 🚀 Функциональность

- Регистрация и вход (локально + через LDAP как пример корпоративной аутентификации).
- Три роли: клиент, мастер, администратор.
- Гибкая система прав (permissions) через связь многие-ко-многим.
- Клиент может:
  - просматривать список мастеров и услуг,
  - создавать запись на выбранное время,
  - просматривать свои записи в личном кабинете.
- Мастер может просматривать свои записи и изменять их статус.
- Администратор управляет пользователями, ролями, мастерами, услугами.
- REST API с проверкой прав (пример: `/api/users` доступен только администратору).
- Полное логирование ошибок и flash-сообщения для пользователя.
- Unit-тесты (pytest) для ключевых моделей и API.

## 🛠 Технологии

- **Python 3.9+**
- **Flask** — микрофреймворк
- **Flask-SQLAlchemy** — ORM
- **Flask-Migrate** — миграции БД
- **Flask-Login** — управление сессиями
- **Flask-WTF / WTForms** — валидация форм
- **Flask-RESTful** — REST API
- **Flask-LDAP** — аутентификация через Active Directory (опционально)
- **pytest** — тестирование
- **Bootstrap 5** — адаптивный интерфейс
- **SQLite** (для разработки) / PostgreSQL (для продакшна)
- **Git, GitHub**
